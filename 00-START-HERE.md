## ⚡ Quick Start (3 Steps)

### 1️⃣ Install Node.js
Download from: **https://nodejs.org/** (v16 or higher)

### 2️⃣ Setup Project
```bash
mkdir mahagenco-dashboard
cd mahagenco-dashboard

# Copy these files here:
# - MAHAGENCO-Master-Dashboard.jsx
# - package.json
# - index.js
# - App.js
# - index.css
# - tailwind.config.js
# - postcss.config.js
# - public/index.html

npm install
```

### 3️⃣ Run Dashboard
```bash
npm start
```

✅ **Dashboard opens at http://localhost:3000**

## 🐛 Common Issues & Fixes

### "npm not found"
→ Install Node.js from https://nodejs.org/

### "Port 3000 in use"
→ Run: `PORT=3001 npm start`

### "Module not found"
→ Run: `npm install`

### "Charts not showing"
→ Check browser console (F12)
